## How to install

**Dependencies**

*Node >= 14.17.0*
<br />
*Yarn >= 1.22.17*
<br />
*NPM >= 6.14.13*

Clone this repository and inside repo run:

```sh 
yarn 
```

or

```sh 
npm install
```

and run:

```sh 
yarn run dev
```

or 


```sh 
npm run dev
```

You can access the using <a href="http://localhost:3000" target="_blank">here</a> : 

```
http://localhost:3000
```
